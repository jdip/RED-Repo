Find what you need to finish the job Choomba.


This site is a quick reference for the Tabletop RPG Cyberpunk RED

Since the project repo is publicly hosted on Github you can fork or download your own copy and modify it as you wish. It will take some technical knowledge and familiarity with HTML, CSS, and Markdown as it is built using Jekyll and hosted on Github Pages.

Feel free to download or fork your own version to modify as you like. I will accept merge requests into the main repo if I subjectively think they fit the purpose of this site and are high quality. This determination will be based solely on my own imperfect opinion.
