Constance City is an alternate setting for the table top role playing game Cyberpunk RED that branches off of our own current timeline instead of following the official lore. It is built using Jekyll for Github Pages.

Feel free to download or fork your own version to modify as you like. I will accept merge requests into the main repo if I subjectively think they fit the established world and are high quality. This determination will be based solely on my own imperfect opinion.
