Net Combat

Net Combat Resolution:

Attacker's Interface + Program ATK + 1d10
or
Black ICE ATK + 1d10
versus
Target's Interface + 1d10
or
Program DEF/Black ICE DEF + 1d10

On a hit the Target suffers the Effect of the Program

Programs are the tools a Netrunner uses to fight, protect, and explore the electronic realm. Activating or Deactivating a Program is a NET Action. Each Program loaded on your Cyberdeck can only be Activated once per Meatspace Round. A Program that is currently Activated (Rezzed) cannot be Activated again until it is Deactivated. Attack programs Deactivate themselves once used.

You can run multiple copies of the same Program on your Cyberdeck if you wish to get around this limitation, and their effects will stack if they are run simultaneously, unless mentioned otherwise.

Installing or Uninstalling a Program takes an hour.

RED Repo is an interactive GM screen for the table top role playing game Cyberpunk RED. It is built using Jekyll for Github Pages.

Feel free to download or fork your own version to modify as you like. I will accept merge requests into the main repo if I subjectively think they fit the purpose of this site and are high quality. This determination will be based solely on my own imperfect opinion.
