Description

Cyberarm Option A One-Handed Light, Medium, or Heavy Melee Weapon (that need not be concealable before its installation) is installed in a Cyberarm so that it can be successfully concealed without a roll, and can be drawn and stowed without an Action. While the weapon is "popped up," the user can't hold anything in this arm's hand. Requires a Cyberarm and takes 2 Option Slots.

RED Repo is an interactive GM screen for the table top role playing game Cyberpunk RED. It is built using Jekyll for Github Pages.

Feel free to download or fork your own version to modify as you like. I will accept merge requests into the main repo if I subjectively think they fit the purpose of this site and are high quality. This determination will be based solely on my own imperfect opinion.
